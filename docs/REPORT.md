# 나침반 (NACHIMBAN) - 최종 보고서

## AI 기반 교육 공공데이터 활용 고등학교 진로 매칭 서비스

---

## 1. 문제 정의 및 배경

### 1.1 문제 인식

현재 중학생의 고등학교 선택과 진로 결정 과정에는 다음과 같은 문제가 존재합니다.

- **정보 비대칭**: 전국 2,407개 고등학교의 교육과정, 개설 과목, 진학 실적 등의 정보가 여러 기관에 산재되어 있어 학생·학부모가 통합적으로 비교하기 어렵습니다.
- **획일적 진로 검사**: 기존 MBTI는 16가지 유형으로만 분류하여, 경계값에 있는 학생들의 개인차를 반영하지 못합니다.
- **주관적 의사결정**: 진로 선택이 데이터가 아닌 주변 평판이나 막연한 인식에 의존하는 경우가 많습니다.
- **교육 격차**: 진로상담 인프라가 부족한 농어촌·소외 지역 학생은 양질의 진로 정보에 접근하기 어렵습니다.

### 1.2 해결 목표

교육 공공데이터를 AI로 통합 분석하여, 누구나 무료로 이용할 수 있는 맞춤형 진로 매칭 서비스를 구현합니다.

## 4. AI 모델 설계

### 4.1 MBTI 연속 스펙트럼 모델

기존 MBTI의 한계를 극복하기 위해, 각 축을 0~100 연속값으로 처리합니다.

```
각 축 4단계 표현:
  0~25  : 강한 첫번째 (강한E, 강한S, 강한T, 강한J)
  26~50 : 약한 첫번째 (약한E, 약한S, 약한T, 약한J)
  51~75 : 약한 두번째 (약한I, 약한N, 약한F, 약한P)
  76~100: 강한 두번째 (강한I, 강한N, 강한F, 강한P)
```

이를 통해 4축 × 4단계 = 256가지 이상의 개인차를 반영하며, 실제로는 연속값으로 무한한 조합이 가능합니다.

### 4.2 직업별 MBTI 프로필 산출 방법

워크넷 공공데이터의 직업별 능력/지식 중요도 점수를 MBTI 4축에 매핑했습니다.

| MBTI 축 | E/S/T/J 방향 (낮은 값) | I/N/F/P 방향 (높은 값) |
|---------|----------------------|---------------------|
| E-I | 설득, 협상, 말하기, 서비스지향 | 논리분석, 추리력, 집중, 기술분석 |
| S-N | 장비조작, 정교한동작, 품질관리 | 창의력, 추리력, 설계, 범주화 |
| T-F | 논리분석, 수리력, 문제해결 | 서비스지향, 사람파악, 가르치기 |
| J-P | 시간관리, 모니터링, 재정관리 | 창의력, 행동조정, 학습전략 |

산출 공식:
```
축 값 = (두번째 방향 평균 점수) / (첫번째 방향 + 두번째 방향) × 100
```

### 4.3 매칭 알고리즘

**진로 추천 (시나리오 1)**
```
최종 적합도 = Holland 코사인 유사도 × 0.6 + MBTI 유클리드 유사도 × 0.4

Holland 유사도: 학생 6축 벡터 ↔ 직업 6축 벡터의 코사인 유사도
MBTI 유사도: 1 - (유클리드 거리 / 최대 거리) × 100
```

**고등학교 매칭 (시나리오 2)**
```
학교 점수 = 교과목 커버리지(40%) + 학교 유형 보너스(5%) + 근접도 보너스(8%)

교과목 커버리지 = (개설된 필수과목 수 / 전체 필수과목 수) × 100
근접도: 같은 동 +8점, 같은 구 +3점
```

## 5. 데이터 분석 결과 (인사이트)

### 5.1 전국 고등학교 현황

- 전국 2,407개 고등학교 중 일반고 1,645개(68.3%), 특성화고 490개(20.4%), 특목고 160개(6.6%), 자율고 112개(4.7%)
- 경기도(505개)가 가장 많고, 세종(22개)이 가장 적음
- 학교별 평균 개설 과목 수는 약 70개이며, 학교 간 편차가 큼

### 5.2 직업 분포 특성

- 462개 직업 중 연구직/공학기술직(89개, 19.3%)이 가장 많고, 농림어업직(10개, 2.2%)이 가장 적음
- Holland 1순위 유형별: 현실형(R) 37%, 진취형(E) 17%, 사회형(S) 15% 순

### 5.3 MBTI 스펙트럼 패턴 (대분류별)

- 연구직/공학직: I 성향(EI=55), T 성향(TF=44) → 내향적+논리적
- 교육/사회복지직: E 성향(EI=46), F 성향(TF=57) → 외향적+공감적
- 예술/디자인직: N 성향(SN=60), P 성향(JP=56) → 직관적+유연적
- 설치/정비/생산직: S 성향(SN=43), I 성향(EI=52) → 감각적+독립적

## 6. 서비스 사용자 플로우

```
[Step 1] 목표 입력 (선택)
  - 희망 고등학교/대학/학과/직업 입력 (자동완성)
  - 건너뛰기 가능

[Step 2] Holland 흥미 검사 (18문항)
  - 6유형 × 3문항, 5점 척도

[Step 3] MBTI 스펙트럼 검사 (32문항)
  - 4축 × 8문항, 7점 척도
  - 결과: 각 축 0~100 연속값

[Step 4] 추가 정보
  - 성별, 거주지(시도/시군구/동), 학년

[결과 대시보드]
  - 종합 의견
  - 프로필 시각화 (Holland 레이더 + MBTI 스펙트럼 바)
  - 목표 매칭도 비교 (입력한 경우)
  - AI 추천 진로 Top 5 (직업 설명 + 적합도 + 이유)
  - 추천 고등학교 3~10개 (유형별 정보 차등 제공)
  - 교육과정 로드맵 (고1~고3)
```

## 7. 기대효과 및 사회적 가치

### 7.1 기대효과

| 대상 | 효과 |
|------|------|
| 학생 | 데이터 기반 객관적 진로 탐색, 적합 학교 발견 |
| 학부모 | 자녀 특성에 맞는 교육 선택 근거 제공 |
| 교사 | 진로지도 보조 자료, 교육과정 설계 참고 |
| 정책 | 지역별 교육 인프라 격차 파악 근거 |

### 7.2 사회적 가치

1. **교육 형평성**: 진로상담 인프라 부족 지역 학생에게도 동일한 AI 추천 제공
2. **정보 접근성**: 산재된 공공데이터를 하나의 서비스로 통합
3. **개인 맞춤화**: 256가지+ 유형 조합으로 획일적 진로지도 탈피
4. **공공데이터 활용 확산**: 교육 분야 공공데이터 활용 모범 사례
5. **의사결정 지원**: 감이 아닌 데이터 기반 진로 선택 문화 조성
